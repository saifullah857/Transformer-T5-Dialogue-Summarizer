## 🔍 Overview

This project has two parts:

| Part | Description |
|------|-------------|
| 📓 **Notebook** | Fine-tune `t5-small` on SAMSum dataset using HuggingFace `Trainer` API |
| 🌐 **Web App** | Serve the trained model via FastAPI + Bootstrap 5 frontend |

The model performs **abstractive summarization** — it generates new text rather than copying sentences, closely mimicking how a human summarizes a conversation.

| Feature | Detail |
|---------|--------|
| 🤖 Base Model | `t5-small` (60M parameters) |
| 📚 Dataset | SAMSum Corpus (messenger-style dialogues) |
| 🏋️ Task | Abstractive Dialogue Summarization |
| 🧮 Framework | HuggingFace Transformers + PyTorch |
| 🌐 Backend | FastAPI + Uvicorn |
| 🎨 Frontend | Bootstrap 5 + Vanilla JS |
| 💻 Device | Auto-detected: CUDA → MPS → CPU |

The **SAMSum** dataset contains realistic messenger-style conversations annotated with human-written summaries.

## ⚙️ Model & Training Configuration

### Training Pipeline (Text_Summarizer.ipynb)

**Step 1 — Load & Sample Data**
```python
train_data = pd.read_csv("samsum-train.csv").sample(n=4000, random_state=42)
val_data   = pd.read_csv("samsum-validation.csv").sample(n=500, random_state=42)
```

**Step 2 — Clean Text**
```python
def clean_data(text):
    text = re.sub(r"\r\n", " ", text)   # remove line breaks
    text = re.sub(r"\s+",  " ", text)   # collapse whitespace
    text = re.sub(r"<.*?>", " ", text)  # strip HTML tags
    return text.strip().lower()
```

**Step 3 — Tokenize**
```python
tokenizer = T5Tokenizer.from_pretrained("t5-small")
# Input  (dialogue) → max_length = 512 tokens
# Target (summary)  → max_length = 150 tokens
```

**Step 4 — Train**

## 🌐 FastAPI Web App

### How It Works

```
Browser (templates/index.html)
      │
      │  POST /summarize/  { "dialogue": "..." }
      ▼
FastAPI (app.py)
      │
      ├─ clean_text()          ← strip noise
      ├─ tokenizer()           ← T5Tokenizer encode
      ├─ model.generate()      ← beam search (num_beams=4)
      └─ tokenizer.decode()    ← decode token ids
      │
      │  { "summary": "...", "time_taken": "1.8s", ... }
      ▼
Browser (displays result)
```

### Key Features of `app.py`

- **`DialogueInput` schema** — Pydantic model with configurable `max_length`, `min_length`, `num_beams`
- **`FileResponse`** — serves `templates/index.html` without Jinja2 (avoids `{}` conflicts with JS)
- **`ThreadPoolExecutor`** — runs heavy inference in a thread pool so FastAPI stays async-responsive
- **In-memory cache** — repeated inputs return instantly (max 100 entries, auto-evicts oldest)
- **Model warmup** — dummy inference on startup prevents slow first real request
- **CORS middleware** — allows requests from any origin

### 8. Run Custom Inference

```python
from transformers import T5Tokenizer, T5ForConditionalGeneration

tokenizer = T5Tokenizer.from_pretrained("./saved_summary_model")
model     = T5ForConditionalGeneration.from_pretrained("./saved_summary_model")

dialogue = "summarize: John: Are we meeting today? Sarah: Yes, at 3pm. John: Perfect, see you then!"
inputs   = tokenizer(dialogue, return_tensors="pt", max_length=512, truncation=True)
outputs  = model.generate(**inputs, max_length=60, num_beams=4)

print(tokenizer.decode(outputs[0], skip_special_tokens=True))
# → "John and Sarah are meeting at 3pm today."
```

## ⚡ Performance Optimizations

| Optimization | Effect |
|---|---|
| `model.eval()` | Disables dropout → faster, consistent inference |
| `torch.no_grad()` | Skips gradient computation → saves memory |
| `torch.set_num_threads(4)` | Uses multiple CPU cores |
| `model.half()` | FP16 on CUDA GPU → ~2x faster inference |
| `torch.compile()` | PyTorch 2.0 graph optimization (auto-skipped if unavailable) |
| `ThreadPoolExecutor` | Non-blocking async inference |
| `run_in_executor` | Keeps FastAPI event loop free during inference |
| In-memory cache | Instant response for repeated inputs (max 100 entries) |
| Model warmup on startup | Eliminates slow first request |
| `repetition_penalty=2.0` | Prevents repeated phrases in output |

## 🔧 Troubleshooting

| Error | Cause | Fix |
|-------|-------|-----|
| `ModuleNotFoundError: transformers` | Library not installed | `pip install transformers torch sentencepiece` |
| `HFValidationError` on model path | Google Colab path used locally | Use correct local path to `saved_summary_model/` |
| `Internal Server Error` on `/` | Wrong template path | Ensure `templates/index.html` exists |
| `NameError: DialogueInput` | Class defined after route | Move class definition before `app = FastAPI(...)` |
| `AssertionError: directory or env` | Empty Jinja2 directory | Use `FileResponse("templates/index.html")` instead |
| Slow first request | Model not warmed up | Warmup block in `app.py` handles this automatically |
| `pip not recognized` | Python not in PATH | Use `python -m pip install ...` |
